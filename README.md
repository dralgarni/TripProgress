# Trip Progress

MVP iPhone app for showing trip completion as a clear percentage.

## Features

- SwiftUI iPhone interface.
- Core Location for current location updates.
- MapKit search and route calculation.
- Progress Mode: 0% to 100%.
- Countdown Mode: 100% to 0%.
- Remaining distance and ETA display.
- Temporary restoration of the last selected trip.
- MVVM structure ready for future CarPlay and Live Activity support.

## Open

Open `TripProgress.xcodeproj` in Xcode, select an iPhone simulator or device, and run.

Location permission is required. Real movement is best tested on device or with a simulated GPX route.
